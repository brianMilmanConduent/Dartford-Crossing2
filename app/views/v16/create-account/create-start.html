Dartford crossing account: sign in or set up

You can set up a personal or business account. If yours is a personal account, you can also apply for a local resident discount.

If you already have an account you can sign in.

Before you start

You'll need to have:

  • an email address to receive a code by email
  • a mobile phone to receive a code by text message
  • the vehicle registration (number plate) you want to add to the account
  • a valid debit card or bank account details if you want to set up bank transfer

  • your vehicle V5C log book
  • your drivers licence
  • a recent utility bill
